# A Comparative Study of Machine Learning Approaches for House Price Prediction

This project is submitted as part of the Machine Learning and Predictive Analytics coursework. It applies and compares three regression models to predict residential property prices using a real-world dataset from Washington State, USA.

## Project Structure

```
MACHINE-LEARNING/
├── EDA.py
├── Model.py
├── data.csv
├── data_cleaned.csv (generated by EDA.py)
├── Img/ (generated graphs)
└── README.md
```

## Dataset
- **Source:** Kaggle (Washington State house sales, 2014–2015)
- **Original size:** 4,600 records, 18 features
- **After cleaning:** 4,545 records, 60 features (target = sale price)

## Data Preparation
- Removed records with invalid or extreme prices
- Removed structural outliers (e.g. unrealistic bedroom counts)
- Engineered new features: property age, size ratios, room counts
- Converted the city feature into a numerical format suitable for modelling

## Models Used
| Role | Model |
|------|-------|
| Primary (chosen) | Random Forest |
| Comparison | Decision Tree |
| Rejected | k-Nearest Neighbours |

## Evaluation Metrics
- R² (coefficient of determination)
- RMSE (Root Mean Squared Error)
- MAE (Mean Absolute Error)
- 5-fold cross-validation

## Results
| Model | R² | RMSE | MAE |
|-------|-----|------|-----|
| Random Forest | 0.7177 | $202,730 | $115,956 |
| Decision Tree | 0.5836 | $246,169 | $144,112 |
| k-NN | 0.6770 | $210,081 | $120,297 |

**Random Forest** was selected as the best model based on its performance and feature interpretability. **k-NN** was rejected due to the curse of dimensionality and lack of interpretability.
